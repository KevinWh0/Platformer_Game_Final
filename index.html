ASCII Platformer

by: Kevin Jillissen

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I wanted to make something unique so I decided to settle on a text based game, as at the time I had been looking into that sort of thing.

2. What I Learned

I need to make the instructions more clear. I made it more clear of what to do. I made it so the map resets when the player wins. They told me to leave the level as a text file so they could make their own maps.

3. Architecture of My Game

The goal of this game is to get to the golden circle as fast as you can avoiding obstacles and other things that might slow you down. You try get the lowest score you can. I use the runner class to start the game, and keep it running, it is also the home to the menu. I use the player class to display the player, and store the player data. The MainGame class is used to display all the world tiles on the screen. Then Enemy class is used to display, and make the enemies move towards to player. So the player interacts with the world. The enemies interact with the world and player. The menu is separate, but also serves as a runner class.

4. Technical Implementation

I kept tweaking the games gravity. I ran into problems where rendering a screen of text made the FPS go down a lot, so I decided to convert some of them into shapes, and this solved the problem.

5. Most Challenging Part

I had problems with gravity, and falling through the ground. also making it so the enemies position converted to player positions to see if they touched.

6. Future Fixes and Updates

I would make more levels, a level select, and possibly a built in level editor. I would make it so the tiles connect better
